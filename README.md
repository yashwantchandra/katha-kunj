## About

"कथा कुंज" is a handcrafted collection of original Hindi stories — short fiction, novel excerpts, and literary vignettes. Each story is written to be read in one sitting, in a typography and palette designed for calm, attentive reading.

## Stories in this collection

1. **आखिरी चिट्ठी** — a letter arrives fifty years late
2. **नदी के उस पार** — novel excerpt; two villages separated by a broken bridge
3. **चाँद का टुकड़ा** — a little girl tries to catch moonlight in a jar
4. **बनारस की गलियाँ** — a woman returns home after twelve years
5. **चाय की आख़िरी घूँट** — forty years of marriage told through a teacup

## Tech

- Single-file static site (`index.html`) — no build, no server
- Noto Serif Devanagari & Tiro Devanagari Hindi typography via Google Fonts
- Light / dark reading mode

## Local preview

Double-click `index.html` — it opens in any browser.

## Adding a new story

Open `index.html`, find the `const stories = [...]` array, and append a new object following the same shape. Commit and push — GitHub Pages redeploys automatically.
